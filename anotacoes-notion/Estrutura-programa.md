# 🏗️ Estrutura de um programa

---

- Input - Process - Output
  - Input: momentos de interação com o usuário
  - Process: verificações e ações para realizar o pedido do usuário
  - Output: são comandos de saída para retornar alguma coisa ao usuário.
- Features (funcionalidades):
  - Diferentes tarefas ou funções que o aplicativo pode realizar.
  - Exemplo: em um aplicativo de chamar carro, features seriam:
    - Solicitar veículo
    - Cadastrar usuário
    - Cadastrar motorista
