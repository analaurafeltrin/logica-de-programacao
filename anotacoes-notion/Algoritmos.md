# 📐 Algoritmos

---

- Algoritmo é como uma sequência de passos para resolver um problema.
- Tudo que precisamos fazer no dia-a-dia exige lógica.
- Por exemplo, para fazer uma receita de bolo, existem alguns passos lógicos (separar os ingredientes, selecionar a forma, colocar os ingredientes, misturar todos…)
- Quando o assunto é programar, é preciso definir as etapas do que deve ser feito.
- Portanto, o algoritmo é a definição clara de cada uma dessas etapas para resolver um problema com o auxílio do computador.
